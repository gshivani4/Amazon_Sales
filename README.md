## Overview

This repository contains an in-depth analysis of Amazon's sales performance, conducted using Excel pivot tables. ## Data Sources

The data used in this analysis was obtained from Amazon's internal order records, including detailed information on sales, orders, and shipping statuses.

## Methodology

1. **Data Cleaning and Processing**:
   - Removed duplicates and irrelevant entries.
   - Addressed missing values.
   - Standardized data formats for consistency.

2. **Data Analysis**:
   - Used Excel pivot tables for dynamic data analysis.
   - Generated visualizations to identify key trends and patterns.

3. **Visualization**:
   - Created a comprehensive dashboard to present key insights.

## Key Insights

- **Order vs. Sales Trends**: Sales peaked in April and experienced significant fluctuations in the following months.
- **Order Status**: 93.99% of orders were shipped successfully, demonstrating efficient order processing.
- **Top Performing States**: Maharashtra leads in sales, followed by Karnataka and Telangana.
- **Amazon vs. Merchant Sales**: Amazon accounts for 69.53% of total sales, with third-party merchants contributing 30.47%.
- **Courier Efficiency**: The majority of orders are shipped promptly, with only 5.72% remaining unshipped.
- **Regional Order Breakdown**: Uttar Pradesh has the highest order count, followed by Karnataka and Haryana.
